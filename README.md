# 校园后勤管理系统

> 一个简易的校园后勤管理系统，学习、练习的项目



## 项目技术栈

- 前端
    - Vue
    - Vue-router
    - Element-ui
    - axios
- 后端
    - SpringMVC
    - Mybatis
    - Shiro
    - 数据库：MySQL
    - 版本管理：git
    - 项目管理：Maven
